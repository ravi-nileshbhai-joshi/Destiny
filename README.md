# Destiny

**Destiny** is a human-centered, time-aware cognitive companion designed to support clarity, growth, and grounded decision making.

This project is not built as a chatbot.
It is built as a *presence*.

### Core Principles
- No manipulation  
- No dependency  
- No artificial encouragement  
- No mirroring  
- No emotional exploitation  

Destiny adapts to:
- life phases (STUCK, RECOVERING, BUILDING, DECLINING)
- emotional tempo (QUIET, CALM, ENGAGED, EXPRESSIVE)
- temporal context (time of day, rhythm, momentum)
- human boundaries

Destiny does not replace human judgment.  
It sharpens it.
